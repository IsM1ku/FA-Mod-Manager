#ArcadeScript

/*
This script defines the available tracks in the UI arcade flow. 

Application:
============
Once the player has selected all the game settings, the goals for each track are searched. The goal 
set whose settings (i.e. laps, direction, car class) most closely match the user's choices will be 
used to determine the merit values. This means that only one set of goals is required per track event
to ensure that there are always some reasonable merit values assigned in game.

DDA handles added to ArcadeScript 12/08/05 - Notes by JH
	Defaults added to Usage example below.
	used the same way as in EventScript, only with an extra lead-in number that is the difficulty level.
	0 = rookie (easy) 	-- note also this calls AI difficulty 3 from difficultyscript.txt
	1 = veteran (medium)    -- note also this calls AI difficulty 6
	2 = master (hard)	-- note also this calls AI difficulty 9


Usage:
======
Theme <Id: Circuit | PointToPoint | DownAndBack | Rampage | LapKnockout>
{
  Item
  {
    Track = <String: track batch name>
    Goal
    {
      DDAValueKeys 	= [0 0.75 1.0 1.065]
      DDADistanceKeys 	= [0 -312 -32 32 256]
//spacer for legibility
      DDAValueKeys 	= [1 0.85 1.05 1.08]
      DDADistanceKeys 	= [1 -312 -32 32 256]
//spacer for legibility
      DDAValueKeys 	= [2 0.95 1.1 1.10]
      DDADistanceKeys 	= [2 -312 -32 32 256]
//spacer for legibility
      GoldGoal   { FinishPosition = 1; DamageTotal = 50000 }
      SilverGoal { FinishPosition = 2; DamageTotal = 40000 }
      BronzeGoal { FinishPosition = 3; DamageTotal = 0 } 
      Laps = <Id: Number>
      Direction = <Id: Forward | Reverse>
      Class = <Id: A | B | C | D | S>
      GoldGoal 
      {
        FinishPosition = <Number> 
        TakedownTotal  = <Number>
        TimeLimit      = <Number>
        DamageTotal    = <Number>
      }
      SilverGoal 
      {
        FinishPosition = <Number> 
        TakedownTotal  = <Number>
        TimeLimit      = <Number>
        DamageTotal    = <Number>
      }
      BronzeGoal 
      {
        FinishPosition = <Number> 
        TakedownTotal  = <Number>
        TimeLimit      = <Number>
        DamageTotal    = <Number>
      }            
    }
  }
}

*/
