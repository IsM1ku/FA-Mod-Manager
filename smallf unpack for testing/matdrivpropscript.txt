// This file contains the drag and traction properties of in-game materials.
// The format should be:
//
// <MATERIAL_NAME> = [<TRACTION_MODIFIER> <DRAG_MODIFIER>]
// 
// Traction -- numbers bigger than 1 are more traction, less than 1 are less traction. 
// Be careful with reducing traction -- losses are amplified at speed... 
//	
// Drag -- numbers bigger than 1 are more drag, less than 1 are less drag.
// I think fairly safe to have very high numbers. 
// UPDATE: High numbers (eg 5+) of drag cause the cars to want to swap ends. 
//
// isOffRoader and isTrackSpecial
// 
// There is also a per car modifier isOffRoader which overrides any traction loss and drag add.
// Sample isOffRoader are Rambler, Kodiak, Honcho, Magnus, Hookzilla
// 
// There is a modifier called isTrackSpecial which adds bonus drag and 
// tractionloss if the material is less than 1.
// Sample isTrackSpecials are Maverick, Wraith, Tec, Streamline, Python
//
// TrackSpecial_traction    finalresult = tractionvalue - 0.05
// TrackSpecial_drag        finalresult = dragvalue * 2.00 
//
// Available materials below as per materialTypes.h
