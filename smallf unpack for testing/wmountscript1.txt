   Format:
   Car <CarId>

     Batch       = "<Batch Name>"
     Name        = "<Verbose Interface Name>"
     Speed       = <Number in [0,1]>
     Handling    = <Number in [0,1]>

     Presets
     {
       <Preset Id 1>
       <Preset Id 2>
       ...
     }
     Colors
     {
       "<Color Name>"
       "<Color Name>"
       ...
     }
     Class = [A,B,C,D,S]
     IsDebugCar
   }
*/

// Setting up presets for cars. 
//IMPORTANT: To use PRESETS in the game: add 'g_useWMountPresets 1' to your pitalocal.psc to activate it.
//Otherwise only the manual loadout system is used.
//Note: Presets automatically OVERRIDE the mount conditionals. So you can make a preset that is not by design. 

/*
Primary means the weapon will be fired with R3 and will be aimable if the weapon supports it. 
Secondary means the weapons will be fired with A and will not be aimable ever. 
Secondary can be a front weapon, and Primary can be a rear weapon. (Not a bug).
MGs, Flamer, Smoke or any weapon that requires a HOLD to fire should only be put in Secondary --> since HOLDING R3 is non ergonomic.
*/

/* List of valid weapons for the weapon presets
The names are Case senstitive!!!
Remember the code names are -1 of actual ingame weapon rank!!: 
(0 == level 1, 1 == level 2, 2 == level 3)
There is no Missiles <---- this will crash the build!!!

Cannon
Missiles
MachineGuns
Rockets
ScatterGun
Grenadier
EmptyWeapon

Minelayer
ReverseScatterGun
ReverseGrenadier
ECMSmoker
EmptyWeapon
*/
