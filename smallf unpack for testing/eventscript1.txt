#EventScript
/*
Usage:

  - There are three kinds of data: id's, strings, and numbers.
    1. Strings are any text contained in quotes (ex: "string")
    2. Id's are text outside of quotes, that begin with a letter, and contain no spaces.
       (ex: Id, ID_1234. But *NOT* 89id, id 1234, "id7231")
    3. Numbers are a set of digits, with the option for decimal values
       (ex: 3, .9, 3.14)
	
  - In the description below, values that can be set by the script writer are contained in < >. The
    type of the data and a simple description of it is provided within the < >.
  - Where there is a set of optional values they are contained in [ ].
  - Where '{' follows directly after some text, this brace and a closing brace '}' is required, 
    and indicates the start of a bunch of information associated with the preceeding text. 
    (ex: Event t121 { ... }. Everything in the braces is information specific to event t121.
  - The example script below only describes the currently commonly used values. You may notice other
    values in existing scripts -- the use of these values is not yet completely decided. Documentation
    on them will be added when the policy on using them is complete.

  Event <id: eventId>
  {
    EventName = <id: event name Id>
    EventType = <id: event type> Race, DownandBack, Rampage, LapKnockout, TimeTrial
    Description = <id: description id>
    DebugText = <string: a debug title for the event>
    TrackName = <id: the track batch name>
    NumLaps   = <number: the number of laps to use>
    Direction = <id: [Forward, Reverse, OpposingForward, OpposingReverse]>
    TimeOfDay = <id: the time of day>
    
    // Vehicles available to the player
    Vehicles
    {
      // Available car
      Car <id: car id -- see wmountscript1.txt>
      {
        // Info on what options the car has -- todo
      }
    }

    // Presets available to the player
    Presets
    {
      // Any number of these
      <id: weapon id -- see wmountscript1.txt>
    }

    // The list of opponents in the race
    Opponents
    { 
      // Describes a single AI
      AI
      {
        Car        = <id: car id -- see wmountscript1.txt>
        Color      = <string: color of the car -- see CarColors.psc>
        Primary    = <id: weapon id -- see wmountscript1.txt>
        Secondary  = <id: weapon id -- see wmountscript1.txt>
        Bounty     = <number: the bounty value for the car. Optional>
      }
      // More AI { ... } may appear
    }

    // Goal conditions for gold status
    GoldGoal
    {
      FinishPosition = <number: The minimum position the player must attain. Optional>
      TakedownTotal  = <number: The minimum takedown count the player must attain. Optional>
      TimeLimit      = <number: The time limit (in seconds) the player must attain. Optional>
      DamageTotal    = <number: The minimum wreck total the player must attain. Optional>
    }
    GoldGoal = RewardName
    SilverGoal
    {
      // Same as GoldGoal
    }
    SilverGoal = RewardName
    BronzeGoal
    {
      // Same as GoldGoal
    }
    BronzeGoal = RewardName
  }
*/
